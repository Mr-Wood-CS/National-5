# Decimal to Binary Conversion

To convert a decimal number into binary, we use the same place value table that was used for binary-to-decimal conversions.

| 128 | 64 | 32 | 16 | 8 | 4 | 2 | 1 |
|------|------|------|------|------|------|------|------|
|.     |.     |.     |.     |.     |.     |.     |.     |

The aim is to build the decimal number using the place values in the table.

---

## Worked Example: Convert 76 to Binary

We are going to convert the decimal number **76** into binary.

Start at the largest value and work towards the right.

For each column, ask yourself:

**Can I subtract this value without going below 0?**

* If the answer is Yes, write a 1 and subtract the value.
* If the answer is No, write a 0 and move on.

---

### Worked Steps

=== "Step 1"

    Can we subtract 128 from 76?

    No, so write **0**.

    76 remains.

=== "Step 2"

    Can we subtract 64 from 76?

    Yes, so write **1**.

    76 - 64 = 12

    We now have **12** remaining.

=== "Step 3"

    Can we subtract 32 from 12?

    No, so write **0**.

=== "Step 4"

    Can we subtract 16 from 12?

    No, so write **0**.

=== "Step 5"

    Can we subtract 8 from 12?

    Yes, so write **1**.

    12 - 8 = 4

    We now have **4** remaining.

=== "Step 6"

    Can we subtract 4 from 4?

    Yes, so write **1**.

    4 - 4 = 0

    We now have **0** remaining.

=== "Step 7"

    Once the remaining value reaches 0, all remaining columns contain **0**.

    Therefore:

      ==**76 = 01001100**==

---

## Quick Check

To convert decimal to binary:

1. Write the place value table.
2. Start at the largest value.
3. If the value fits, write 1 and subtract it.
4. If the value does not fit, write 0.
5. Continue until you reach the final column.

!!! warning "Exam Tip"

      Show your subtraction working beside the table. This helps you gain method marks and makes mistakes easier to spot.
