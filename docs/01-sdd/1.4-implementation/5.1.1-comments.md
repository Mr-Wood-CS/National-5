# Comments

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Identify comments in Python code.
    - Explain why comments are used.
    - Add comments to make code easier to understand.

---

## Program Comments

Comments are lines in a program that are ignored by the computer.

They are used to explain what the code does.

---

## Python Comments

In Python, comments start with `#`.

```python title="Comment Example" linenums="1"
# This is a comment
# The computer ignores this line
```

At National 5, you are expected to use comments in your code.

---

## Summary

| Feature | Purpose |
|---------|---------|
| `#` | Starts a comment in Python |
| Comment | Explains code for human readers |
