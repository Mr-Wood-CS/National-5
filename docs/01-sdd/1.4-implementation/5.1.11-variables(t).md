# Variables: Integer and String

**Task 1**

Create a Python program that registers runners for a race.

Ask the user to enter:

- the runner's name (string)
- their competition number (integer)
- their running club (string)

After all the information has been entered, display each value to check it has been stored correctly.

Example:

```text
Input:
Runner's name: Ava Brown
Competition number: 12
Running club: Langholm Harriers

Output:
Runner's name: Ava Brown
Competition number: 12
Running club: Langholm Harriers
```

**Task 2**

Create a Python program that generates an employee ID card.

Ask the user to enter:

- the worker's name (string)
- their date of birth as numbers only, for example `25061985`
- their job title (string)
- their unique ID number (integer)

Display an employee ID card on the screen.

The card must:

- display **Cyberdyne Systems** at the top
- clearly label each piece of information
- display all of the details entered by the user

For example:

```text
Cyberdyne Systems

Name: Jim Halpert
Date of Birth: 25061985
Job Title: Salesman
Employee ID: 12345
```
