# Input, Output, Variables and Data Types

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Use comments to explain code.
    - Create variables and assign values to them.
    - Choose suitable variable names.
    - Identify and use common data types.
    - Display output using `print()`.
    - Get user input using `input()`.
    - Convert input to a number using `int()` or `float()`.
    - Carry out calculations using arithmetic operators.
    - Join strings together using concatenation.

---

## Overview

Programs need to store data, take in data from the user, process it, and display results.

In Python, these ideas are closely linked:

- variables store values
- data types describe the kind of value being stored
- `input()` lets the user enter data
- `print()` displays data on the screen
- comments explain what your code is doing
