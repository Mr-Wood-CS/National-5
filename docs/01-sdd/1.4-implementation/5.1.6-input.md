# Input

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Get user input using `input()`.
    - Store input in a variable.
    - Use a prompt to tell the user what to type.

---

## Input from the User

Input means allowing the user to enter data into a program.

In Python, input is entered using `input()`.

---

## Getting Text Input

The message inside the brackets is called a prompt.

It tells the user what to type.

```python title="Text Input" linenums="1"
name = input("Please enter your name: ")

print("Hello", name)
```

The data entered by the user is stored in a variable.

---

## Multiple Inputs

A program can ask the user for several pieces of data.

```python title="Multiple Inputs" linenums="1"
title = input("Enter book title: ")
author = input("Enter book author: ")
genre = input("Enter book genre: ")
```

---

## Summary

| Feature | Purpose |
|---------|---------|
| `input()` | Gets data from the user |
| Prompt | Tells the user what to type |
| Variable | Stores the input |
