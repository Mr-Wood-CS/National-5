# Inputting Numbers

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Explain why number input must be converted.
    - Use `int()` to input an integer.
    - Use `float()` to input a real number.

---

## Why Number Input Must Be Converted

Data entered using `input()` is automatically stored as a string.

If the input will be used in a calculation, it must be converted to a number.

---

## Integer Input

Use `int()` when the input should be a whole number.

```python title="Integer Input" linenums="1"
age = int(input("Enter age: "))
```

---

## Real Number Input

Use `float()` when the input should be a real number.

```python title="Real Number Input" linenums="1"
price = float(input("Enter price: "))
```

---

## Summary

| Feature | Purpose |
|---------|---------|
| `input()` | Gets data as a string |
| `int()` | Converts input to an integer |
| `float()` | Converts input to a real number |
