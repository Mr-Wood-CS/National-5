**Task 1**

Type the program below into VS Code and run it.

```python title="Slab Calculator" linenums="1"
# Get information from the user
slabsWide = int(input("How many slabs wide? "))
slabsDeep = int(input("How many slabs deep? "))
costPerSlab = float(input("How much per slab? "))

# Calculate how many slabs needed
totalSlabs = slabsWide * slabsDeep

# Calculate total cost
totalCost = totalSlabs * costPerSlab

# Display the result
print("You require", totalSlabs, "slabs")
print("The total cost is £", totalCost)
```
