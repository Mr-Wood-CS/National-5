## What Is an Array?

An array is a data structure that can store multiple values under one name.

In Python, an array is usually called a list.

```python title="Array Example" linenums="1"
scores = [15, 20, 19, 18, 17]
```

This array stores five test scores.

Using an array is better than creating lots of separate variables such as `score1`, `score2` and `score3`.

---

## Elements and Indexes

Each value in an array is called an element.

The position of an element is called its index.

Array indexes start at `0`.

```python title="Indexes" linenums="1"
scores = [15, 20, 19, 18, 17]
```

| Index | Element |
|-------|---------|
| `0` | `15` |
| `1` | `20` |
| `2` | `19` |
| `3` | `18` |
| `4` | `17` |

---

## Accessing Elements

To access an element, write the array name followed by the index in square brackets.

```python title="Accessing Elements" linenums="1"
scores = [15, 20, 19, 18, 17]

third_score = scores[2]

print(third_score)
```

This program outputs `19`.

---

## Updating Elements

An element in an array can be changed by assigning a new value to an index.

```python title="Updating Elements" linenums="1"
scores = [15, 20, 19, 18, 17]

scores[0] = 16
scores[3] = 20
```

The array now stores:

```python title="Updated Array" linenums="1"
scores = [16, 20, 19, 20, 17]
```

---

## Appending Elements

The `append()` method adds a new element to the end of an array.

```python title="Appending Elements" linenums="1"
scores = [15, 20, 19, 18, 17]

scores.append(14)
```

The array now stores:

```python title="Appended Array" linenums="1"
scores = [15, 20, 19, 18, 17, 14]
```

---

## Summary

| Feature | Purpose |
|---------|---------|
| Array | Stores multiple values under one name |
| Element | One value stored in an array |
| Index | The position of an element |
| `[ ]` | Used to access or update an element |
| `append()` | Adds a new element to the end of an array |
