# Input and Output

**Task 1**

Create a new Python program that prints the lyrics of a song over multiple lines.

Choose any appropriate song. Each line of the lyrics should appear on a separate line of the screen.

---

**Task 2**

Create a Python program that asks the user which school they attend.

Ask the user to enter:

- their school name

The program should then display:

```text
You're a pupil of [school name].
```

For example:

```text
Input:
Greenwood High

Output:
You're a pupil of Greenwood High.
```

---

**Task 3**

Create a Python program that asks the user for their favourite number.

Ask the user to enter:

- their favourite number

The program should then display a message that includes the number they entered.

For example:

```text
Input:
7

Output:
Your favourite number is 7.
```
