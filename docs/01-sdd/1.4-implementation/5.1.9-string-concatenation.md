## Joining Strings

String concatenation means joining strings together.

In Python, strings can be joined using `+`.

```python title="String Concatenation" linenums="1"
word1 = "Hello"
word2 = "World"

sentence = word1 + " " + word2

print(sentence)
```

The space has to be added manually.

Without the space, the output would be `HelloWorld`.

---

## Summary

| Feature | Purpose |
|---------|---------|
| Concatenation | Joins strings together |
| `+` | Joins strings in Python |
| `" "` | Adds a space between joined strings |
