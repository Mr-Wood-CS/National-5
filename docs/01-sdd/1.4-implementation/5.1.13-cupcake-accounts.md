# Assessment One

## Cupcake Accounts

Sue and Dave run a cupcake stall. They want a program to help keep track of their money each week.

A large cupcake always costs £2, and a small cupcake always costs £1. These should be set as constants in the program.

Each week the program should:

- ask how many large cupcakes were sold
- ask how many small cupcakes were sold

The program should then calculate:

- the money made from large cupcakes
- the money made from small cupcakes
- the total money made altogether

Next, the program should ask for the ingredient costs for that week.

The program should then work out:

- the profit, which is total money made minus ingredient costs
- how much money each owner gets when the profit is split equally

Finally, the program should display:

- total money made
- ingredient costs
- profit
- share for Sue
- share for Dave
