## What Is a Variable?

A variable is a named storage location for a value.

Variables allow a program to store data and use it later.

---

## Assigning Values

The `=` symbol is used to assign a value to a variable.

```python title="Assigning Values" linenums="1"
age = 15
name = "Bob"
email = "bob@gmail.com"
```

In computing, variable names should be meaningful.

For example, `age` is clearer than `x`.

---

## Variable Naming Rules

Python variable names must follow these rules:

- A variable name must start with a letter or an underscore.
- A variable name cannot start with a number.
- A variable name can only contain letters, numbers and underscores.
- Variable names are case-sensitive, so `age`, `Age` and `AGE` are different variables.
- A variable name cannot be a Python keyword, such as `if`, `for` or `while`.
