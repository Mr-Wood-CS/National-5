## Arithmetic Operators

Python programs often carry out calculations.

The result of a calculation can be stored in a variable.

| Operator | Meaning | Example |
|----------|---------|---------|
| `+` | Addition | `total = num1 + num2` |
| `-` | Subtraction | `difference = num1 - num2` |
| `*` | Multiplication | `answer = num1 * num2` |
| `/` | Division | `answer = num1 / num2` |
| `**` | Raise to a power | `square = num1 ** 2` |

---

## Calculation Example

```python title="Area Calculation" linenums="1"
length = float(input("Enter length: "))
width = float(input("Enter width: "))

area = length * width

print("The area is", area)
```

| Feature | Purpose |
