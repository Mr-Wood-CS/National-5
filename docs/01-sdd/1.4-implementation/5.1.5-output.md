# Output

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Display text using `print()`.
    - Display values stored in variables.
    - Display text and variables together.

---

## Output with print

Output means displaying information from a program.

In Python, output is displayed using `print()`.

---

## Displaying Text

Text is written inside quotation marks.

```python title="Displaying Text" linenums="1"
print("Hello world")
print("Welcome to National 5")
```

---

## Displaying Variables

To display the value stored in a variable, do not put the variable name in quotation marks.

```python title="Displaying Variables" linenums="1"
name = "Aisha"
age = 15

print(name)
print(age)
```

---

## Displaying Text and Variables

Text and variables can be displayed together by separating them with commas.

```python title="Text and Variables" linenums="1"
name = "Aisha"

print("My name is", name)
```

---

## Summary

| Feature | Purpose |
|---------|---------|
| `print()` | Displays output |
| Quotation marks | Used around text |
| Comma in `print()` | Displays text and variables together |
