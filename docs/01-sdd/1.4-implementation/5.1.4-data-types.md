---

## Common Data Types

Variables store values of different data types.

At National 5, you need to know five common data types.

| Data Type | Description | Example |
|----------|-------------|---------|
| Integer | A whole number | `12`, `-50`, `100` |
| Real number | A number with a decimal point | `22.5`, `0.001` |
| String | Text made from letters, digits or symbols | `"hello"`, `"abc123"` |
| Character | A single letter, digit or symbol | `"a"`, `"Z"`, `"#"` |
| Boolean | A value that is true or false | `True`, `False` |

---

## Python Examples

```python title="Data Type Examples" linenums="1"
# Integer
age = 15

# Real number
price = 0.99

# String
subject = "Computing"

# Character
first_initial = "F"

# Boolean
is_student = True
```

!!! note

    Python does not have a separate `char` data type. A character is stored as a string with one character.
