# Data Types and Structures Tasks

## Before You Start

These tasks will help you practise choosing suitable data types and data structures.

!!! tip "Remember"

    * <strong style="color: #1565c0;">String</strong> - text made from more than one character
    * <strong style="color: #2e7d32;">Integer</strong> - a whole number
    * <strong style="color: #6a1b9a;">Real</strong> - a number with a decimal point
    * <strong style="color: #c62828;">Boolean</strong> - True or False
    * <strong style="color: #ef6c00;">Character</strong> - one letter, number or symbol
    * <strong style="color: #00838f;">Variable</strong> - stores one value
    * <strong style="color: #ad1457;">Array</strong> - stores multiple values

For each question, write your answers in your Google Workbook.

Explain each answer using the data in the question.

## Task 3: Inputs And Outputs

**Question 1**

A supermarket self-scan app reads a number on each product, for example `128761`, and displays the product name.

The customer then presses a button to confirm if that is the correct product.

The app adds the price of the product to their total amount due and displays this at the end of their shop.

<span style="color: #1565c0; font-weight: 700;">Identify</span> three inputs for this program.

For each input, <span style="color: #2e7d32; font-weight: 700;">state</span> the most suitable data type.

!!! hint

    The total amount due is not an input.

___
