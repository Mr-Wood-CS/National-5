# Data Types and Structures Tasks

## Before You Start

These tasks will help you practise choosing suitable data types and data structures.

!!! tip "Remember"

    * <strong style="color: #1565c0;">String</strong> - text made from more than one character
    * <strong style="color: #2e7d32;">Integer</strong> - a whole number
    * <strong style="color: #6a1b9a;">Real</strong> - a number with a decimal point
    * <strong style="color: #c62828;">Boolean</strong> - True or False
    * <strong style="color: #ef6c00;">Character</strong> - one letter, number or symbol
    * <strong style="color: #00838f;">Variable</strong> - stores one value
    * <strong style="color: #ad1457;">Array</strong> - stores multiple values

For each question, write your answers in your Google Workbook.

Explain each answer using the data in the question.

## Task 3: Choose Data Types For A Program

For each data item, ask yourself:

* Is it text?
* Is it a whole number?
* Is it a number with a decimal point?
* Is it True or False?

**Question 1**

A supermarket self-scan app reads a product code, for example `128761`, and finds the product name.

The customer then presses a button to confirm if that is the correct product.

The app adds the price of the product to their total amount due.

<span style="color: #1565c0; font-weight: 700;">Identify</span> four data items this program needs to store or process.

For each data item, <span style="color: #2e7d32; font-weight: 700;">state</span> the most suitable data type.

!!! hint

    Think about the product code, product name, button choice and money values.

___

**Question 2**

An online game asks users to log in with a username and password.

It counts the total number of hours they have spent playing the game.

It stores the player's high scores.

<span style="color: #1565c0; font-weight: 700;">Identify</span>:

* four data items this program needs to store or process
* the most suitable data type for each data item
* one suitable data structure used by this program

<span style="color: #2e7d32; font-weight: 700;">Explain</span> each choice.

___

**Question 3**

A cinema booking app lets a customer choose a film, screening time and number of tickets.

After the customer makes their choices, the app stores:

* the film title
* the screening time
* the number of tickets booked
* whether the booking has been confirmed
* the total cost

<span style="color: #1565c0; font-weight: 700;">Identify</span> the five data items stored by the app.

For each data item, <span style="color: #2e7d32; font-weight: 700;">state</span> the most suitable data type.

!!! hint

    The data type describes the value being stored or processed.

    For example, the number of tickets is an <strong style="color: #2e7d32;">integer</strong>, because it is a whole number.
