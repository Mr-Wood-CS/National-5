# Select, From and Where Tasks

Bishra's Bakery keeps delivery information in a database. Bishra will ask for different reports, and your job is to turn each request into an SQL query.

These tasks practise using:

- `SELECT` to choose the fields to display
- `FROM` to choose the table
- `WHERE` to search for specific records
- `AND` and `OR` to join conditions together

The two tables you will use are:

- `DeliveryDrivers`
- `DeliverySchools`

For each task, write and run the SQL query, then screenshot your SQL code and the results table for your workbook.

## Task One

> "We need all the information about the drivers displayed."

Bishra wants a full list of every delivery driver.

Your query should display every field from the `DeliveryDrivers` table.

## Task Two

> "Can we have all the deliveries but just the school names, towns and addresses set out?"

Bishra wants a shorter delivery list that only shows the location details.

Your query should use the `DeliverySchools` table and display:

- school name
- town
- address

## Task Three

> "Can we have all the information on the Hilton Primary delivery please?"

Bishra wants to look up one specific delivery.

Your query should use the `DeliverySchools` table, display all fields, and only show the record for `Hilton Primary`.

## Task Four

> "We need the names, ages and IDs of all the drivers who are 30 years old or younger."

Bishra wants to find the younger drivers in the bakery.

Your query should use the `DeliveryDrivers` table and display:

- driver name
- age
- employee ID

Only show drivers whose age is `30` or younger.

## Task Five

> "We might be adding extra Inverness schools if we can fit in the deliveries. We need to know all the Inverness deliveries at ten in the morning or after to see what other deliveries we can do. Display the school names, the town and time of delivery of these schools found."

Bishra is checking whether more Inverness deliveries could fit into the morning route.

Your query should use the `DeliverySchools` table and display:

- school name
- town
- delivery time

Only show deliveries where the town is `Inverness` and the delivery time is `10:00` or later.

## Task Six

> "New laws state all drivers under the age of 25 or 35 or over need to be re-tested in their vans for their driving licence each year. We need to find all the drivers affected along with their van registration number to send to the testing centre."

Bishra needs a list of drivers who may need extra licence checks.

Your query should use the `DeliveryDrivers` table and display:

- driver name
- van registration number

Only show drivers who are under `25` or drivers who are `35` and older.
