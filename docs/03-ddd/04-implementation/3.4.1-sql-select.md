# SQL SELECT Operations

!!! info "What You Need to Know"

    For National 5 Computing Science, you should be able to:

    - Retrieve data from a database using SQL.
    - Use `SELECT ... FROM ...` queries.
    - Use `SELECT ... FROM ... WHERE ...` queries.
    - Apply conditional operators to filter records.
    - Use logical operators (`AND` and `OR`) to combine conditions.
    - Sort query results using `ORDER BY`.

---

## SELECT ... FROM ...

The `SELECT ... FROM ...` statement is used to retrieve data from a table.

You can either:

- Display all columns in a table.
- Display only specific columns.

### Display All Columns

```sql title="Select All Columns" linenums="1"
SELECT *
FROM entity_name;
```

The `*` symbol means "all columns".

### Display Specific Columns

```sql title="Select Specific Columns" linenums="1"
SELECT column1, column2
FROM entity_name;
```

This query would display only `column1` and `column2`.

---

## SELECT ... FROM ... WHERE ...

The `WHERE` clause is used to filter records.

Only records that meet the specified condition will be displayed.

```sql title="Select With a Condition" linenums="1"
SELECT column1, column2, column3
FROM entity_name
WHERE column3 > value;
```

**Example**

```sql title="Pupils Aged Over 14" linenums="1"
SELECT Forename, Surname
FROM Pupils
WHERE Age > 14;
```

This query would return only pupils older than 14.

---

## Conditional Operators

Conditional operators are used inside a `WHERE` clause to compare values.

| Operator | Meaning |
|----------|---------|
| `=` | Equal to |
| `>` | Greater than |
| `<` | Less than |
| `>=` | Greater than or equal to |
| `<=` | Less than or equal to |
| `<>` | Not equal to |

**Example**

```sql title="Find Pupils Aged 15" linenums="1"
SELECT *
FROM Pupils
WHERE Age = 15;
```

---

## Logical Operators

Logical operators allow multiple conditions to be combined.

### AND

`AND` requires all conditions to be true.

```sql title="AND Example" linenums="1"
SELECT *
FROM Pupils
WHERE Age > 14
AND House = 'Nevis';
```

This query returns pupils who are older than 14 and belong to Nevis house.

### OR

`OR` requires at least one condition to be true.

```sql title="OR Example" linenums="1"
SELECT *
FROM Pupils
WHERE House = 'Nevis'
OR House = 'Locheil';
```

This query returns pupils from either Nevis or Locheil.

---

## ORDER BY

The `ORDER BY` clause is used to sort query results.

### Ascending Order (ASC)

Ascending order is the default order.

```sql title="Ascending Order" linenums="1"
SELECT Forename, Surname, Age
FROM Pupils
ORDER BY Age ASC;
```

This displays the youngest pupils first.

### Descending Order (DESC)

```sql title="Descending Order" linenums="1"
SELECT Forename, Surname, Age
FROM Pupils
ORDER BY Age DESC;
```

This displays the oldest pupils first.

---

## Summary

| SQL Feature | Purpose |
|------------|---------|
| `SELECT ... FROM ...` | Retrieves data from a table |
| `WHERE` | Filters records |
| Conditional Operators | Compare values |
| `AND` | All conditions must be true |
| `OR` | At least one condition must be true |
| `ORDER BY` | Sorts query results |

These commands form the foundation of SQL queries at National 5 level.
