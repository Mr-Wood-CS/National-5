# SQL UPDATE Operations

!!! info "What you need to Know"

    For National 5 Computing Science, you should be able to:

    - Use SQL commands to update existing records.
    - Understand the purpose of the `UPDATE` statement.
    - Use a `WHERE` clause to identify which records should be changed.
    - Apply update commands to simple database tables.

---

## What is UPDATE?

The `UPDATE` statement is used to change information that already exists in a table.

When updating data, you specify:

1. The table that contains the record.
2. The field that needs to change.
3. The new value.
4. The condition that identifies which record should be updated.

```sql title="Update Syntax" linenums="1"
UPDATE table_name
SET column_name = new_value
WHERE condition;
```

### Example

```sql title="Update a Pupil Age" linenums="1"
UPDATE Pupils
SET Age = 16
WHERE StudentNum = 13;
```

This would change Emma Brown's age to 16.

!!! warning "Important"
    Always use a `WHERE` clause when updating records. Without a `WHERE` clause, every record in the table will be updated.

---

## Summary

| Part | Purpose |
|------|---------|
| `UPDATE` | Chooses the table to change |
| `SET` | Chooses the field and new value |
| `WHERE` | Chooses which record or records should be changed |

The `UPDATE` statement is used when existing data needs to be changed.
