# SQL DELETE Operations

!!! info "What you need to Know"

    For National 5 Computing Science, you should be able to:

    - Use SQL commands to delete existing records.
    - Understand the purpose of the `DELETE` statement.
    - Use a `WHERE` clause to identify which records should be removed.
    - Apply delete commands to simple database tables.

---

## What is DELETE?

The `DELETE` statement is used to remove records from a table.

When deleting data, you specify:

1. The table that contains the record.
2. The condition that identifies which record should be deleted.

```sql title="Delete Syntax" linenums="1"
DELETE FROM table_name
WHERE condition;
```

### Example

```sql title="Delete a Pupil" linenums="1"
DELETE FROM Pupils
WHERE StudentNum = 13;
```

This would remove Emma Brown's record from the table.

!!! warning "Warning"
    Always check your `WHERE` clause carefully. If you omit the `WHERE` clause, all records in the table may be deleted.

---

## Summary

| Part | Purpose |
|------|---------|
| `DELETE FROM` | Chooses the table to remove data from |
| `WHERE` | Chooses which record or records should be removed |

The `DELETE` statement is used when existing data needs to be removed from a database table.
