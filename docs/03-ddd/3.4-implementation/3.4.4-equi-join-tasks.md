## Task One

[:material-database: Open the SQL workspace](https://onecompiler.com/sqlite){: .inline-launch__link target="_blank" rel="noopener" }

Copy and run the SQL below to create the tables.

```sql
CREATE TABLE Pupils
(
    StudentNum int,
    Forename varchar(20),
    Surname varchar(20),
    Age int,
    House varchar(10)
);

CREATE TABLE Guidance
(
    EmployeeNum int,
    Title varchar(8),
    Surname varchar(20),
    House varchar(10)
);
```


Copy each SQL query below, run it, and add a screenshot of your output to your notes.

a) **Display all joined records**

This query joins the `Pupils` and `Guidance` tables using the matching `House` field.

```sql
SELECT *
FROM Pupils, Guidance
WHERE Pupils.House = Guidance.House;
```

___

b) **Display selected fields from both tables**

This query displays each pupil's name and their guidance teacher's surname.

```sql
SELECT Pupils.Forename, Pupils.Surname, Guidance.Surname
FROM Pupils, Guidance
WHERE Pupils.House = Guidance.House;
```

___

c) **Display selected fields with clearer headings**

This query uses aliases to make the output easier to read.

```sql
SELECT Pupils.Forename AS PupilForename,
       Pupils.Surname AS PupilSurname,
       Guidance.Title AS TeacherTitle,
       Guidance.Surname AS TeacherSurname
FROM Pupils, Guidance
WHERE Pupils.House = Guidance.House;
```
