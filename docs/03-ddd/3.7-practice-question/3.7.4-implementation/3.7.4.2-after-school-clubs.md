# After School Clubs

Use this page to practise the SQL skills needed for National 5 Computing Science.

The tasks are based on a small database about school clubs. Some questions ask you to display data, while others ask you to add, update or delete records.

For questions that change the database, a short test query is included. Run the test query after your answer to check whether your SQL has worked.

## Practice

You should practise:

- `SELECT ... FROM`
- `WHERE`
- comparison operators such as `=`, `<`, `>`, `<=` and `>=`
- `AND` and `OR`
- `ORDER BY`
- `INSERT`
- `UPDATE`
- `DELETE`
- simple equi-joins using primary and foreign keys

**Question 8**
Add a new club with these details:

| clubID | clubName | category | day | room |
|---:|---|---|---|---|
| 8 | Netball Club | Sport | Wednesday | Gym |

To test your answer, run:

```sql
SELECT *
FROM Club
WHERE clubID = 8;
```

If your query has been successful, the test query should show Netball Club.

---

**Question 9**
Change Football Club so that it is held in the Gym instead of the Field.

To test your answer, run:

```sql
SELECT clubName, room
FROM Club
WHERE clubName = 'Football Club';
```

If your query has been successful, the room should be Gym.

**Question 10**
Remove the member called Harris Young from the database.

To test your answer, run:

```sql
SELECT *
FROM Member
WHERE forename = 'Harris'
AND surname = 'Young';
```

If your query has been successful, the test query should return no records.
