# After School Clubs

Use this page to practise the SQL skills needed for National 5 Computing Science.

The tasks are based on a small database about school clubs. Each section shows an example first, then gives you a similar question to try yourself. Your task is shown in the **Task** box.

[:material-database: Open the SQL workspace](https://onecompiler.com/sqlite/44t3abhh6){: .inline-launch__link target="_blank" rel="noopener" }

You should practise:

- `SELECT ... FROM`
- `WHERE`
- comparison operators such as `=`, `<`, `>`, `<=` and `>=`
- `AND` and `OR`
- `ORDER BY`
- `INSERT`
- `UPDATE`
- `DELETE`
- simple equi-joins using primary and foreign keys

**Question 1**

The example displays every field from the `Club` table.

```sql
SELECT *
FROM Club;
```

!!! question "Task"

    Display every field from the `Member` table.

---

**Question 2**

The example displays only the club names and days.

```sql
SELECT clubName, day
FROM Club;
```

!!! question "Task"

    Display only the forename, surname and year group of each member.

---

**Question 3**

The example finds the club held on Tuesday.

```sql
SELECT *
FROM Club
WHERE day = 'Tuesday';
```

!!! question "Task"

    Display every field for clubs in the Computing category.

---

**Question 4**

The example finds members in S4 or above.

```sql
SELECT forename, surname, yearGroup
FROM Member
WHERE yearGroup >= 4;
```

!!! question "Task"

    Display the forename and surname of members in S3.

---

**Question 5**

The example finds Computing clubs held in room C12.

```sql
SELECT clubName, day, room
FROM Club
WHERE category = 'Computing'
AND room = 'C12';
```

!!! question "Task"

    Display the club name and room for clubs in the Computing category that are held on Friday.

---

**Question 6**

The example finds clubs held on Monday or Friday.

```sql
SELECT clubName, day
FROM Club
WHERE day = 'Monday'
OR day = 'Friday';
```

!!! question "Task"

    Display the forename, surname and year group of members who are in S3 or S5.

---

**Question 7**

The example displays clubs in alphabetical order by club name.

```sql
SELECT clubName, category, day
FROM Club
ORDER BY clubName ASC;
```

!!! question "Task"

    Display the members' forenames, surnames and year groups from oldest year group to youngest year group.

---

**Question 8**

The example adds a new club.

```sql
INSERT INTO Club
VALUES (7, 'Debate Club', 'Communication', 'Tuesday', 'B2');
```

!!! question "Task"

    Add a new club with these details:

    | clubID | clubName | category | day | room |
    |---:|---|---|---|---|
    | 8 | Netball Club | Sport | Wednesday | Gym |

To test your answer, run:

```sql
SELECT *
FROM Club
WHERE clubID = 8;
```

If your query has been successful, the test query should show Netball Club.

---

**Question 9**

The example changes the room for Chess Club.

```sql
UPDATE Club
SET room = 'A5'
WHERE clubName = 'Chess Club';
```

!!! question "Task"

    Change Football Club so that it is held in the Gym instead of the Field.

To test your answer, run:

```sql
SELECT clubName, room
FROM Club
WHERE clubName = 'Football Club';
```

If your query has been successful, the room should be Gym.

---

**Question 10**

The example removes Art Club.

```sql
DELETE FROM Club
WHERE clubID = 6;
```

!!! question "Task"

    Remove the member called Harris Young from the database.

To test your answer, run:

```sql
SELECT *
FROM Member
WHERE forename = 'Harris'
AND surname = 'Young';
```

If your query has been successful, the test query should return no records.

---

**Question 11**

The example joins the two tables to display each member with their club.

```sql
SELECT forename, surname, clubName
FROM Member, Club
WHERE Member.clubID = Club.clubID;
```

!!! question "Task"

    Display each member's forename, surname, club name and club day.

---

**Question 12**

The example joins the two tables and filters for Computing clubs.

```sql
SELECT forename, surname, clubName
FROM Member, Club
WHERE Member.clubID = Club.clubID
AND category = 'Computing';
```

!!! question "Task"

    Display the forename and surname of every member who attends a Sport club. Sort the results by surname in alphabetical order.
