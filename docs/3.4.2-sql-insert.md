# SQL INSERT, UPDATE and DELETE Operations

!!! info "What you need to Know"

    For National 5 Computing Science, you should be able to:

    - Use SQL commands to add records to a table.
    - Use SQL commands to update existing records.
    - Use SQL commands to delete records.
    - Understand the purpose of the `INSERT`, `UPDATE` and `DELETE` statements.
    - Apply these commands to simple database tables.

---

## INSERT Operations

The `INSERT` statement is used to add a new record to a table.

When inserting data, you specify:

1. The table you want to add the record to.
2. The columns that will receive data.
3. The values to be stored.

```sql title="Insert Syntax" linenums="1"
INSERT INTO table_name (column1, column2, column3)
VALUES (value1, value2, value3);
```

### Example

```sql title="Add a New Pupil" linenums="1"
INSERT INTO Pupils (StudentNum, Forename, Surname, Age)
VALUES (13, 'Emma', 'Brown', 15);
```

This would add a new pupil record to the `Pupils` table.

---

## UPDATE Operations

The `UPDATE` statement is used to change information that already exists in a table.

You must use a `WHERE` clause to identify which record should be updated.

```sql title="Update Syntax" linenums="1"
UPDATE table_name
SET column_name = new_value
WHERE condition;
```

### Example

```sql title="Update a Pupil Age" linenums="1"
UPDATE Pupils
SET Age = 16
WHERE StudentNum = 13;
```

This would change Emma Brown's age to 16.

!!! warning "Important"
    Always use a `WHERE` clause when updating records. Without a `WHERE` clause, every record in the table will be updated.

---

## DELETE Operations

The `DELETE` statement is used to remove records from a table.

A `WHERE` clause is normally used to specify which record should be deleted.

```sql title="Delete Syntax" linenums="1"
DELETE FROM table_name
WHERE condition;
```

### Example

```sql title="Delete a Pupil" linenums="1"
DELETE FROM Pupils
WHERE StudentNum = 13;
```

This would remove Emma Brown's record from the table.

!!! warning "Warning"
    Always check your `WHERE` clause carefully. If you omit the `WHERE` clause, all records in the table may be deleted.

---

## Summary

| SQL Command | Purpose |
|------------|---------|
| `INSERT` | Adds a new record to a table |
| `UPDATE` | Changes an existing record |
| `DELETE` | Removes a record from a table |

These three commands allow you to maintain and modify the data stored within a database.
