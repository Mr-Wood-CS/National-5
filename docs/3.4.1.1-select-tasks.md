## Task Two

Use the `Guidance` table for the questions below.

**Build the table**

Copy and run this SQL to create the table.

```sql
CREATE TABLE Guidance
(
    EmployeeNum int,
    Title varchar(8),
    Surname varchar(20),
    Age int,
    House varchar(10)
);
```

**Add the data**

Copy and run this SQL to insert the data.

```sql
INSERT INTO Guidance
(EmployeeNum, Title, Surname, Age, House)
VALUES
(1, 'Mr', 'Stewart', 28, 'Nevis'),
(2, 'Miss', 'Kelly', 45, 'Lochiel'),
(3, 'Mr', 'MacDonald', 57, 'Rannoch'),
(4, 'Mrs', 'Jordan', 32, 'Shiel');
```

a) Write the SQL code to select all of the information to be displayed.

b) Write the SQL code that will show all the people with the title `Mr`.

c) Write the SQL code that will show all the staff in the house `Rannoch`.

d) Write the SQL code that will show everyone aged between `20` and `40`.

e) Write the SQL code that will show everyone who is greater than or equal to `40` years old and is in house `Lochiel`.

f) Write the SQL code that will show the `Title`, `Surname` and `Age` of people who are aged less than `35` in descending order.

g) Write the SQL code that will show the `Title`, `Surname` and `House` of everyone in the database.
